Description

This project We use instagram api to take popular and searching tag data.We showed top 9 popular picture and if user search special tag, searched pictured can will be seen. When cursor is top the image, we will information about this picture like who share, how many times liked and what is image text.When clicked the image, user connect the instagrams clicked image pages.

Description of Data

  • We get data via ajax and it's data type is jsonp.Data contains;
    • Image's Username
    • Image's Text
    • Image's Link
    • Image's Liked Count

Installation

  • Api Authentication
